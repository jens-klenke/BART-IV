\section{\texorpdfstring{\texttt{R} als
Taschenrechner}{R als Taschenrechner}}\label{r-als-taschenrechner}}

Öffnen Sie \texttt{RStudio} auf Ihrem Rechner und machen Sie sich mit
der Oberfläche vertraut. Versuchen Sie unter dem Menüpunkt \emph{File}
ein \texttt{R}-Skript zu öffnen und anschließend die Funktionen der
unterschiedlichen Fenster zu verstehen.

\begin{enumerate}[label=\alph*)]
  \item Benutzen Sie R als Taschenrechner: Berechnen Sie $3 + 4$, $6 - 8$, $3 \times 5$ und $10 \div 3$.
  \item Speichern Sie das letzte Ergebnis ab, indem Sie diesem den Namen $x$ geben. Nutzen Sie hierfür \verb|<-| den Zuweisungsoperator in \verb|R|.
  \item Lassen Sie sich das Ergebnis ausgeben.
  \item Runden Sie das Ergebnis $x$ auf $2$ Nachkommastellen: \verb|round(x, 2)|.
\end{enumerate}
